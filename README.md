# Zen Portfolio

A personal portfolio website showcasing my game development projects and programming skills.

live link: https://zenportfolio.netlify.app/

## About

This is my professional portfolio featuring:
- Interactive particle background effect
- Project showcase with detailed case studies
- Skills, education, and experience sections
- Contact form for inquiries

## Built With

- **HTML5** - Structure and content
- **CSS3** - Styling and responsive design
- **JavaScript** - Interactivity and animations
- **Particles.js** - Background particle effects

## Projects Featured

1. **Aura** - Top-down RPG built with Unreal Engine 5 and C++ using GAS
2. **Rum Runner's Revenge** - TIGER Award-nominated 2D action RPG (Unity, C#)
3. **3D Graphics Simulation** - Real-time rendering with OpenGL (C++)
4. **Kitchen Chaos** - Fast-paced cooking game (Unity, C#)
5. **Dungeon Slayer** - Current project: wave-based action RPG (UE5, C++)

## Features

- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Interactive Project Panels** - Detailed information slides in on click
- **Particle Effects** - Dynamic animated background
- **Contact Form** - Integrated email functionality via FormSubmit
- **Smooth Navigation** - Scroll-to-section navigation with smooth transitions

## Local Setup
